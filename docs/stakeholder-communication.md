# 📣 Stakeholder Communication Model

## Цель

Обеспечить прозрачную коммуникацию между бизнесом, IT и руководством.

---

## Форматы коммуникации

| Формат | Частота | Участники |
|-------|---------|----------|
| Weekly sync | Еженедельно | Команда |
| Steering committee | Ежемесячно | Руководство |
| Incident updates | По событию | Stakeholders |
| Quarterly review | Ежеквартально | Все функции |

---

## Принципы

- Кратко и по сути  
- Фокус на рисках и решениях  
- Прозрачные статусы  
- Ответственные за follow-up
